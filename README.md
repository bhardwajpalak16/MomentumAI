# 🚀 MomentumAI

> **Your Last-Minute AI Productivity Coach**

An AI-powered productivity assistant that helps students and professionals plan their day, recover from missed deadlines, and stay productive using Google Gemini AI.

---

## 🌐 Live Demo

🔗 https://momentumai-31a47.web.app

---

## 🎥 Project Overview

MomentumAI intelligently analyzes your tasks and generates:

- 📅 AI Daily Planner
- 🚨 Rescue Plans for missed deadlines
- 🧠 AI Productivity Coach
- 📊 Productivity Analytics
- 🔥 Momentum Score & Streak Tracking

---

# ✨ Features

### 🤖 AI Planner
- Generates optimized daily schedules
- Prioritizes tasks intelligently
- Estimates completion time

### 🚨 Rescue Engine
- Detects overdue tasks
- Creates recovery plans
- Suggests realistic timelines

### 🧠 AI Productivity Coach
- Personalized motivational coaching
- Daily productivity tips
- Smart suggestions

### 📊 Analytics Dashboard
- Task completion statistics
- Productivity trends
- Momentum Score
- Current Streak

### 🔐 Authentication
- Google Sign-In using Firebase Authentication

### ☁️ Cloud Database
- Firestore stores:
  - Tasks
  - User Stats
  - Momentum
  - Streaks
